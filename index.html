📊 订阅转换命令

# 解析订阅为 Clash 格式
python3 sub-convert.py --url "你的订阅链接" --format clash

# 关键词过滤
python3 sub-convert.py --url "你的订阅链接" --keyword "US" --format clash

📋 分流规则使用指南

Clash 的分流规则从上到下逐一匹配，第一个命中的规则决定走哪个出口。

推荐规则顺序

# 1. 广告拦截（节省流量）
- DOMAIN-SUFFIX,doubleclick.net,REJECT

# 2. 国内直连（节省代理流量）
- DOMAIN-SUFFIX,cn,DIRECT
- DOMAIN-SUFFIX,baidu.com,DIRECT
- IP-CIDR,10.0.0.0/8,DIRECT
- GEOIP,CN,DIRECT

# 3. 流媒体优先匹配
- DOMAIN-SUFFIX,netflix.com,Netflix
- DOMAIN-SUFFIX,youtube.com,YouTube

# 4. 默认代理
- MATCH,Proxy

❓ 常见问题

Q1: 节点延迟低但速度慢是怎么回事？▼
延迟(ping)≠带宽(speed)。可能是：机场入口带宽小、晚高峰多人使用、机场限速策略。建议同时测试多个节点的带宽表现。
Q2: 如何判断机场是否跑路前兆？▼
🚨 警惕：突然无法访问官网、无法充值、客服消失、节点数量急剧减少、价格异常涨价。长用机场建议保留备用方案。
Q3: Clash Premium 和 Clash Meta 有什么区别？▼
Clash Meta（MetaCubeX维护）支持更多新协议（Hy2/TUIC），社区活跃，规则支持脚本扩展。推荐使用 Clash Verge 或 Stash（基于 Meta）。
